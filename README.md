
# Defect Detection System (OpenCV + PyTorch)

Prototype repository to detect scratches/defects on metal surfaces using OpenCV for preprocessing and PyTorch for training/inference.
Includes:
- Synthetic data generator
- PyTorch UNet model (small) for segmentation-first prototype
- Training and inference scripts
- ONNX export + quantization scripts
- FastAPI inference endpoint
- Streamlit dashboard for local monitoring
